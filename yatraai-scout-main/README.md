# YatraAI Scout Pro

AI-powered travel companion designed for Indian heritage landmarks and navigation.

## Core Features
- **🔍 AI Landmark ID**: Identify monuments using TensorFlow ML models.
- **🗺️ Map Explorer**: Interactive routing and POI search using OSM.
- **🌦️ Weather Intel**: 7-day deep forecast.
- **💬 Scout AI**: Chat with history and local hotel suggestions.
- **💰 Smart Budget**: Track and visualize expenses with Firebase.

## Setup
- `npm install`
- Configure `.env` with Firebase and Groq keys.
- `npm run dev`
